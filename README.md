# ShadeC-EVO

This is a Shader Collection for Conitec 3D Gamestudio A8

It adds features such as:
-Deferred Rendering;
-Bloom;
-Depth of Field;
-SSAO & SSDO;
-Softparticles;
-Realtime Shadows;
-Refractions;
-Easy to customize Ubershader.
